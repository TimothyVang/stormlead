# StormLead Testing Evidence

This folder stores local proof that the app/browser surface was exercised.

- `screenshots/` - explicit screenshots saved by Playwright tests.
- `videos/` - copied Playwright videos from the latest evidence run.
- `logs/` - command output and Playwright JSON results.
- `playwright-report/` - HTML report generated by Playwright.
- `playwright-artifacts/` - raw Playwright traces, screenshots, and videos.

Latest browser smoke target: `GET /admin` from the `ping-post` FastAPI app.

The browser smoke mocks backend JSON calls so the admin UI can be reviewed without requiring the Docker Postgres stack.
